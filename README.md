# 📚 Biblioteca Digital em Python

## 📌 Sobre o projeto
Sistema para gerenciar documentos digitais de uma biblioteca.

## ⚙️ Funcionalidades
- Adicionar documentos
- Listar documentos por tipo
- Remover documentos
- Renomear documentos

## 🧠 Tecnologias
- Python 3
- Manipulação de arquivos (os)

## ▶️ Como executar
python3 main.py
