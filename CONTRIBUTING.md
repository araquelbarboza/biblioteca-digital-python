# Guia de Contribuição

## Como executar o projeto

```bash
python3 main.py
```

## Como realizar alterações

1. Faça as modificações necessárias no código.
2. Salve os arquivos.

## Como criar um commit

```bash
git add .
git commit -m "Descrição da alteração"
```

## Como enviar para o GitHub

```bash
git push
```

## Como criar um Pull Request

1. Envie suas alterações para uma branch.
2. Acesse o repositório no GitHub.
3. Clique em "Compare & pull request".
4. Descreva as alterações realizadas.
5. Envie o Pull Request para revisão.
